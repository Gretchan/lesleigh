# Cosmic Love

A small interactive night-sky web app with a moon, stars, and hidden love notes.

## How it works
Each clickable star opens a memory or message.
One final star reveals a special note.

## Customize
Open `index.html` and edit the `memories` array.

You can change:
- title
- subtitle
- star messages
- final hidden message

## Deploy
This is a static site, so it can be deployed easily on:
- Vercel
- Netlify
- GitHub Pages

## Quick deploy
1. Push this repo to GitHub
2. Import it into Vercel or Netlify
3. Deploy
4. Send the live link
